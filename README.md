# viser-web-direct — Gaussian Splatting 웹 뷰어

브라우저에서 바로 도는 3D Gaussian Splatting 뷰어/에디터입니다.
React 19 + Three.js + @react-three/fiber 기반, WASM(SIMD) 깊이 정렬,
서버 없이 로컬 파일·CDN 테스트 씬을 로드할 수 있고 viser 서버 연동(스냅샷/델타 스트리밍)도 지원합니다.
PWA로 설치할 수 있으며, 전체 상태를 URL 하나로 공유할 수 있습니다.

## 실행

```bash
npm install
npm run dev      # 개발 서버
npm run build    # 프로덕션 빌드 (tsc + vite)
npm test         # 라운드트립 테스트 (cov / ply / npz / splat / spz)
```

URL 플래그: `?jssort` — WASM 대신 JS 정렬기 강제 (디버깅용).

## 지원 포맷

| 포맷 | 비고 |
| --- | --- |
| `.ply` | 3DGS 표준 PLY. SH degree-1(f_rest 0–8) 색상 지원 |
| `.splat` | antimatter15 32B/splat 포맷 |
| `.spz` (β) | Niantic spz v2/v3, gzip 컨테이너. SH1 지원. 실제 캡처 파일 검증 전이라 β |
| 서버 npz | viser 서버 스냅샷/델타 스트리밍 |

파일은 "PLY/SPLAT/SPZ 열기" 버튼 또는 드래그&드롭으로 로드합니다.
대용량 씬은 설정의 **로드 축소**(1:2 / 1:4)로 로드 시점에 서브샘플링할 수 있습니다.

### 테스트 씬

메뉴의 테스트 드롭다운에서 Train / Truck / Plush / Bicycle / Garden / Stump / Treehill을
Hugging Face CDN에서 스트리밍 로드합니다. 첫 방문 시 Train이 지정 카메라로 자동 로드됩니다.

## 조작

| 입력 | 동작 |
| --- | --- |
| 좌클릭 드래그 / 한 손가락 드래그 | 트랙볼 회전 (화면 기준) |
| 휠 / 핀치 | 전진 비행 줌 (씬을 뚫고 지나감) |
| 우클릭 드래그 / 두 손가락 평행이동 | 패닝 |
| 두 손가락 비틀기 | 화면 축 롤 |
| W A S D (+Shift 가속) | 비행 이동 |
| 더블클릭 / 더블탭 | 가우시안 점 선택 (Shift: 누적/토글) |
| 길게 누르기 (550ms) | 해당 지점을 회전축(피벗)으로 지정 |

감도는 설정에서 **회전 / 확대 / 이동** 세 개로 조절합니다 (마우스·터치 공통).

## 도구

- **측정** — 점을 찍어 폴리라인 거리 측정
- **다면체 선택** — 가우시안에 고정된 3D 정점 4개 이상으로 볼록 다면체를 만들어 내부 가우시안 일괄 선택, 영역 저장/재적용 가능
- **주석(📌)** — 3D 위치에 텍스트 노트 (공유 링크에 포함)
- **크롭 박스** — AABB 바깥 잘라내기
- **플로터 정리** — 고립 스플랫 제거
- **비교** — 다중 런 오버레이, 편집 전/후 A/B 와이프, 색상 diff 히트맵
- **공전** — 현재 피벗을 중심으로 카메라 자동 궤도
- 선택 반전/확장, 선택만 남기기, 선택을 회전축으로, 색/투명도/크기 편집, undo/redo (메모리 상한 설정 가능)

## 성능·품질

- **품질 프리셋** 5단계 (매우낮음–최고) — 아래 항목들을 일괄 설정
- 적응형 DPR: fps가 최소 fps(기본 15) 아래로 내려갈 때만 해상도를 단계적으로 낮춤
- 거리 LOD(확률적 시닝 + 면적 보정), 정렬 임계값, LOD 비율, MSAA 토글
- 렌더 설정(블러, 컬링, 알파 테스트, SH 색상 등)은 antimatter15/splat 기본값과 동일
- 모든 설정은 localStorage에 저장되어 새로고침 후에도 유지

## 공유

**공유** 버튼은 현재 상태 전부를 `?z=` 파라미터 하나에 담은 URL을 만듭니다:
씬 소스(CDN 테스트 씬 또는 서버 런), 카메라 포즈, 렌더 설정, 화면 표시 옵션,
3D 주석, 녹화된 카메라 경로. 받은 사람은 동일한 화면을 보게 됩니다.
(선택 상태와 로컬 파일은 공유에서 제외됩니다.)

통계 패널에서 카메라 포즈를 `{"p":[…],"t":[…]}` JSON으로 복사하고,
같은 형식을 붙여넣어 해당 시점으로 즉시 이동할 수 있습니다.

## 기타

- **북마크** — 카메라 시점을 썸네일과 함께 저장
- **최근 씬** — 최근 로드한 테스트 씬/런 바로가기
- **PWA** — 브라우저 설치 지원, 정적 자산 오프라인 캐시
- **내보내기** — 선택 영역 .ply 내보내기, PNG 프레임 시퀀스(zip) 내보내기

## 구조

```
src/
  App.tsx                  앱 셸, 로딩/편집/공유/패널
  RenderSettings.ts        렌더 설정 정의 + 기본값
  components/              씬 오브젝트, 제스처 컨트롤, 패널 UI
  splat/                   가우시안 렌더러, 셰이더, 정렬 워커(WASM/JS)
  lib/                     포맷 파서(ply/splat/spz/npz), 수학, URL 상태
tests/                     포맷 라운드트립 테스트
```

내부 표현은 가우시안당 32바이트 패킹 버퍼(위치 f32×3 · 그룹 u32 · 공분산 f16×6 · RGBA u8×4)이며,
편집은 copy-on-write로 이루어집니다.
